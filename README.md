# 汤姆猫餐厅 · Vue 展示项目

这是一个将原始单页餐厅展示改为 **Vite + Vue 3** 的项目。

## 启动方式

1. 安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
3. 浏览器访问：
   ```bash
   http://localhost:5173/
   ```

## 项目结构

- `index.html`：Vite 入口 HTML
- `src/main.js`：Vue 应用入口
- `src/App.vue`：页面主结构
- `src/styles.css`：全局样式
- `src/components/`：各页面区块组件
